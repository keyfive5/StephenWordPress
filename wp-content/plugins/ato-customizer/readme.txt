Custom sticker & label customizer for All Take Out (Kew Stick Inc.).

== Description ==

Adds to any WooCommerce store:

* Per-product configuration: templates, materials, sizes, shapes and quantity tiers (with per-tier pricing) — set up in a product's "ATO Customizer" tab.
* A drag-and-drop label design editor (Fabric.js): text with a curated font library, image upload, clipart, generated QR codes, background colours, layers, undo/redo, and a shape-accurate cut line.
* Designs saved with a unique reference (e.g. ATO-8F3K2A), attached to the cart line and the order.
* VIP membership: every new customer account (including accounts created at checkout) gets the VIP role, 50 complimentary stickers credited, free ground shipping, and a "VIP Benefits" tab in My Account.
* Production emails: when an order is paid, the admin receives a spec sheet — configuration, fonts used, uploaded image / QR counts, preview image and a link to the design.
* Admin review panel ("ATO Designs"): browse every customer design, open it in the same editor, refine it, and keep a visible edit history.

== Installation ==

1. Install and activate WooCommerce.
2. Upload and activate this plugin.
3. Edit a product, open the "ATO Customizer" tab, tick "Enable customizer" and fill the option lists.
4. For payments, install "WooCommerce Stripe Gateway" and connect the client's Stripe account.
